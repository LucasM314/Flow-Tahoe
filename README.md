## ⚙️ Optimal Settings

For the best experience and the most authentic look, it is recommended to configure Flow Launcher with the following settings:

### Fonts
- **Query Box & Result Title:** `SF Pro Regular`
- **Result Subtitle:** `SF Pro Text Regular`

> 💡 *Don't have the fonts? You can download them from [this repository](https://github.com/sahibjotsaggu/San-Francisco-Pro-Fonts).*

### General Preferences

| Preference | Recommended Setting |
| :--- | :--- |
| **Import Theme Size** | ✅ Apply |
| **Search Window Position** | Use the reference image above |
| **Always Preview** | ❌ Disable |
| **Home Page** | ❌ Disable |
| **Fixed Window Size** | ✅ Enable (Lock the width using the reference image above) |
| **Maximum Results** | `6` to `7` |
| **Show Placeholder** | ❌ Disable |

### Clock & Date Widget

For the cleanest look, enable **only** the clock *or* the date, but not both.
